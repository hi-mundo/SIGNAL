# SIGNAL Contributors

This file is reserved for contributors, maintainers, reviewers, translators, profile authors, and research contributors.

SIGNAL uses an upstream-first model. Contributions should preferably be submitted to the canonical repository so the community can benefit from a shared framework.

## Contribution categories

Contributors may be recognized for:

- framework design;
- criteria improvements;
- pattern contributions;
- anti-pattern contributions;
- research notes;
- benchmark mappings;
- retrieval overlap maps;
- domain profiles;
- translations;
- examples;
- governance improvements;
- issue reports;
- critical review.
